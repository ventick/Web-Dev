# Online Store Lab 5

Angular project for Lab 5. This version continues the Lab 4 store and adds a category hierarchy, a three-component architecture, like buttons, delete buttons, and parent-child communication through inputs and outputs.

## Run locally

```bash
npm install
npm start
```

Open `http://localhost:4200/`.

## Build

```bash
npm run build
```

## What is implemented

- 4 categories with 5 products each
- Category -> product hierarchy
- `AppComponent`, `ProductListComponent`, and `ProductItemComponent`
- `input()` and `output()` communication between components
- Like and delete functionality
- Real Kaspi product links and images
- WhatsApp and Telegram share buttons
